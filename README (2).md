\# Mutual Fund Analytics



\## Overview

Mutual Fund Analytics project developed as part of the Bluestock Capstone Program.



\## Technologies

\- Python

\- Pandas

\- SQLite

\- Jupyter Notebook

\- Power BI

\- Git \& GitHub



\## Datasets

10 mutual fund datasets including NAV history, fund performance, investor transactions, AUM, SIP inflows and portfolio holdings.



\## Features

\- Data Cleaning

\- SQLite Database

\- Fund Performance Analytics

\- Risk Metrics (VaR, CVaR)

\- Fund Recommendation System

\- Power BI Dashboard



\## How to Run



\### Install Dependencies

pip install pandas numpy matplotlib sqlalchemy



\### Run ETL

python run\_pipeline.py



\### Open Dashboard

dashboard/bluestock\_mf\_dashboard.pbix
